CoreQml.h - Core qml register types

Register all enabled GenesisX QML modules on engine. More...

Header: #include <CoreQml.h>
Since: Qt 6.10

Detailed Description

Via CoreQml.h you link the different modules. Which modules are registered depends on your qmake QT, e.g. genesisx_app_notifications, genesisx_app_ab, etc.

code snippet setting modules list in qmake:
GX_LOADED_MODULES = $$QT
GX_LOADED_MODULES_CSV = $$join(GX_LOADED_MODULES, ",")
DEFINES += GX_LOADED_MODULES=\\\"$$GX_LOADED_MODULES_CSV\\\"
And set in main.cpp.
GXCore::registerEnabledQmlModules(&engine, GX_LOADED_MODULES);
Optionally you can load everything by setting.
GXCore::registerEnabledQmlModules(&engine);
