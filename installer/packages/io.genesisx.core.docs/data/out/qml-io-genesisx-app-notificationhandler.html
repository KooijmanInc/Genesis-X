Property Documentation

initialized : bool

True if the user has initialized notifications


Signal Documentation

initializedChanged()

Emitted whenever the initialized property changes.

Note: The corresponding handler is onInitializedChanged.


notificationReceived(string title, string body, var data)

Emitted when a notification is received.

The title and body contain the notification text. The data map may include extra key–value pairs from the payload.

Note: The corresponding handler is onNotificationReceived.


tokenChanged(string token)

Emitted when the push-notification token changes. The new token is provided in token.

Note: The corresponding handler is onTokenChanged.


Method Documentation

Initializes the handler with the given options map The options argument can contain platform-specific settings such as initial topics or configuration flags.
