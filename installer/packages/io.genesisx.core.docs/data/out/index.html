Genesis-X Documentation

Introduction

Genesis-X is a modular, high-performance application framework designed for Qt 6. It provides production-ready tooling, fully integrated developer utilities, and advanced modules to accelerate development for desktop, mobile, and embedded platforms.

Genesis-X is created and maintained by Kooijman Incorporate Holding B.V., with a focus on real-world developer experience, QML productivity, and clean modular extensibility.

This documentation provides an overview of all Genesis-X modules and includes detailed C++, QML, tooling, and workflow references.

Features

Genesis-X contains several key modules:

  • Core Fundamental utilities, logging, configuration, and the Genesis-X application environment.
  • Utils Cross-platform helpers, platform abstractions, and additional C++ convenience APIs.
  • Orm A lightweight, high-level ORM layer with support for HTTP controllers, request handling, response objects, and future database connectors.
  • Assets (QML Design System) (In development) A QML-based UI toolkit with components, typography, spacing, color tokens, and layout helpers.
  • Physics (In development) A modern physics extension intended for Qt Quick 3D View3D applications.
  • Snippets (Developer Tools) Qt Creator snippets for faster QML and C++ development.
  • Wizard Templates Project templates integrated into Qt Creator’s “New Project” wizard.

Getting Started

To begin using Genesis-X, include the desired modules in your qmake project:

QT += genesisx
QT += genesisx_app_notifications
QT += genesisx_assets
QT += genesisx_orm        # optional
QT += genesisx_physics    # in development

In your main.cpp add:

#include <GenesisX/CoreQml.h>

int main(int argc, char* argv[])
{
    QApplication app(argc, argv);

    GXCore::registerEnabledQmlModules(&engine);
}

This will register the qmltypes of the chosen features.

In QML, import modules using:

import GenesisX.Notifications 1.0

Note: If you select a feature like genesisx_app_notifications the you don't have to add the feature genesisx, this is always included when selecting a feature.

Refer to the Getting Started Guide for installation steps, module usage, and editor integration.

Developer Tools

Genesis-X ships with several productivity tools:

  • Qt Creator Snippets Auto-completion helpers for common QML / C++ patterns.
  • Qt Creator Wizard Templates Create new Genesis-X applications directly from File > New Project.
  • QDoc / Qt Assistant Integration This documentation is auto-registered in Qt Creator when the installer is run.
