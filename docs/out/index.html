Genesis-X SDK

What is Genesis-X?

Genesis-X is a general-purpose SDK for Qt/C++ projects. It bundles common mobile/desktop features behind a single, consistent API so you can ship faster across platforms.

Current features

  • Notifications — Local and push notifications with a unified API. See the C++ class gx::NotificationHandler for details.

What’s next

  • Remote API — A client API for talking to your backend server (authentication, messaging, data sync).
  • Symfony server template — A ready-to-use Symfony package that pairs with Genesis-X for an end-to-end setup (routes, DTOs, auth, push webhooks). This will be published as a separate package you can deploy or extend.

Quick start (Notifications)

C++

#include <GenesisX/NotificationHandler.h>

gx::NotificationHandler notifier;
notifier.requestPermission();
notifier.showLocal(QStringLiteral("Hello from Genesis-X"));

QML

import GenesisX.Notifications

NotificationHandler {
    onMessageReceived: (m) => console.log(m.title, m.body)
}
